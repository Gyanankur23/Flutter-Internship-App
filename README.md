# Flutter Internship App

## Vision
A simple Flutter app scaffold created for the Mobile App Development with Flutter Virtual Internship (Week 1 Deliverable).

## Objectives
- Provide a clean starting point for app development.
- Demonstrate GitHub setup and version control usage.
- Lay the foundation for adding UI screens in Week 2.

## Navigation Flow
- Login Screen
- Home Dashboard
- Program Listing
- Program Details/Profile
